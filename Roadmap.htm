The roadmap to your career success

You’re in control. We’re with you along the way.

  • Step 1. Select your career path and register on our website.

    If you are looking for a quick and fun introduction to GitHub, you’ve found it. This class will get you started using GitHub in less than an hour.

  • Step 2. Meet with a career mentor to work out an action plan for your next goal

    Learn how to use GitHub Desktop to copy a repository to your computer, track changes and communicate with GitHub.

  • Step 3. Your mentor will keep track of your progress towards your career goal.

    Learn how to use the Command Line to copy a repository to your computer, track changes and communicate with GitHub.

  • Step 4. Attending professional development courses to learn the hard skills you will need

    Learn how to configure your local git environment to work with GitHub and how to complete GitHub flow using the GitHub Extension for Visual Studio.

  • Step 5. Your mentor will look for potential job/intership opportunities for you

    Quick guides to git out of sticky situations.

  • Step 6. When you close to complete your action plan, your mentor lodges job applications for you.

    Learn how to create seamless integrations by directly accessing and updating GitHub data using GraphQL.

  • Step 7. You attend interviews and wait for the outcome

    An entire suite of courses that will get you up and running with Electron.
